# AD LABS

**Tech | Automação | IA**

Bem-vindo ao repositório central do **AD LABS**. Este projeto consolida iniciativas de automação, inteligência artificial e desenvolvimento tecnológico.

## Estrutura do Projeto

- **00_SETUP**: Configurações iniciais e scripts de ambiente.
- **01_DOCS**: Documentação, checklists e registros.
- **02_PROMPTS**: Biblioteca de prompts para IAs.
- **03_MIDIA**: Ativos visuais e mídia.
- **04_CODIGO**: Código fonte de projetos e automações.
- **05_AUTOMACAO**: Fluxos de trabalho (N8N, Make, etc.).
- **99_ARQUIVO**: Histórico e arquivos arquivados.

## Objetivo

Centralizar o desenvolvimento e a gestão de conhecimento para soluções de tecnologia avançada.
